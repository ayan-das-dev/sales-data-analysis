# SQL Data Analysis Project

## 📌 Overview

This project demonstrates SQL-based data analysis on an e-commerce dataset.
It covers basic to advanced queries including aggregation and window functions.

---

## ⚙️ Tools Used

* SQL

---

## 📂 Project Structure

* data/ → Raw dataset
* setup/ → Table creation and data insertion scripts
* queries/ → SQL queries for analysis

---

## 🔄 Queries Covered

### 🟢 Level 1 (Basic)

* Total sales per city
* Total sales per category
* Average order value per city
* Number of orders per customer

### 🟡 Level 2 (Intermediate)

* Top 3 customers by spending
* City with highest sales
* Product with highest revenue
* Total discount per category

### 🔴 Level 3 (Advanced - Window Functions)

* Customer ranking using RANK()
* Running total using SUM() OVER
* Top order per city using ROW_NUMBER()
* Percentage contribution of category
* Repeat customers
* Previous vs current order using LAG()

---

## 📊 Key Concepts

* GROUP BY and Aggregation
* Window Functions (RANK, ROW_NUMBER, LAG)
* Analytical SQL
